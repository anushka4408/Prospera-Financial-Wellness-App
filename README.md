## 🎯 Overview

<img src="https://media.giphy.com/media/3oKIPEqDGUULpEU0aQ/giphy.gif" width="100%" />

* 📊 Track expenses
* 🔮 Predict budgets
* 🧠 Analyze financial health
* 🤖 Get AI-driven recommendations

Built using **MERN + Google Gemini AI**
